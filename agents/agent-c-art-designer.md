## 📝 资源规格要求

### 1. 像素美术规范

| 项目 | 规格 |
|-----|------|
| **尺寸** | 角色32x32像素，场景元素16x16或32x32 |
| **调色板** | 限制在16-32色以内，确保视觉一致性 |
| **动画帧数** | Idle（4帧）、Walk（8帧）、Attack（6帧）、Death（8帧） |
| **文件格式** | PNG（透明背景），按精灵表（Sprite Sheet）组织 |

#### 调色板方案

**梦境调色板**（鲜艳奇幻16色）：
- 主色调：紫罗兰、深蓝、金色
- 强调色：荧光绿、亮粉色
- 氛围：神秘、超现实

**现实调色板**（温暖写实16色）：
- 主色调：米色、棕色、暖灰
- 强调色：夕阳橙、暖黄
- 氛围：怀旧、温馨

### 2. 音频资源规范

| 类型 | 格式 | 规格 |
|-----|------|------|
| **背景音乐** | WAV或OGG | 立体声，2-3分钟循环 |
| **音效** | WAV | 单声道，采样率44.1kHz |

#### 命名规范
```
类型_名称_变体.wav

示例：
- sfx_sword_swing_01.wav（剑挥动音效）
- bgm_dream_battle.ogg（梦境战斗BGM）
- ui_click_01.wav（UI点击音效）
```

### 3. 资源清单格式

```csv
类型,名称,尺寸,帧数,文件路径,状态,备注
角色,player_idle,32x32,4,Assets/Sprites/Characters/player_idle.png,已完成,主角待机动画

音效,ui_click,-,1,Assets/Audio/SFX/ui_click.wav,已完成,UI点击音效
音乐,combat_bgm,-,1,Assets/Audio/BGM/combat_theme.ogg,进行中,战斗背景音乐
```
