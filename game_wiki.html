概念设定

理论框架

🌑 核心概念定义

影子（Shadow）

定义与属性

定义：依靠光而存在的共生种族，寄生在人类宿主身上

光依赖性：影子的存在以光为前提，在绝对黑暗中会逐渐失去自我意识

寄生性：必须依附于宿主（人类）才能长期存在

意识性：拥有独立的思维和情感，是智慧生命体

梦境连接性：在宿主入睡时可以脱离并进入梦境世界

存在状态

  • 依附态：白天寄生在宿主身上，跟随宿主的行动
  • 游离态：夜晚进入梦境世界，独立活动
  • 消逝态：在黑暗中停留过久，逐渐失去自我

梦境世界（Dream World）

世界特性

主观性：世界形态受梦主情绪和经历影响

流动性：物理法则不稳定，空间结构可变

象征性：事物往往具有隐喻和象征意义

多层性：可能存在梦中梦的嵌套结构

运作原理

  • 情绪投射：现实中的情绪会具象化为梦境中的实体
  • 记忆重构：记忆碎片重组为梦境场景
  • 欲望具现：潜意识欲望以象征形式呈现

情绪投射（Emotional Projection）

投射机制

基于弗洛伊德的投射理论，将内在情绪外化为外部实体

心理学基础：防御机制的一种表现形式，潜意识处理情绪的方式

投射类型

情绪类型 投射表现 典型形态
恐惧 威胁性实体 怪物、陷阱
孤独 虚空、缺失 空洞、迷宫
愤怒 破坏性力量 狂暴敌人
悲伤 沉重、缓慢 沼泽、锁链
焦虑 压迫感 追逐、倒计时

⚙️ 系统原理

共生系统原理

寄生-宿主关系模型

能量流动：宿主 → 影子

信息反馈：影子 → 宿主

情感共鸣：双向影响

能量交换

  • 影子从宿主获取生命能量维持存在
  • 影子在梦境中的成长反馈给宿主
  • 双方情绪状态互相影响

共生平衡

  • 健康的共生关系：双方互惠
  • 失衡状态：一方过度索取
  • 断裂风险：长期分离导致连接减弱

历史演变

远古时期

太阳永恒照耀，无黑夜。影子一族繁荣发展，建立独立文明和社会结构。

太阳衰败

黑夜诞生，大部分影子失去意识，沦为普通影子。少数影子学会寄生求生。

现代

寄生成为影子生存的主要方式。影子与人类文明融合，共生关系成为常态。

昼夜双模式原理

模式切换机制

入睡切换

触发条件：梦主进入睡眠状态

切换过程：意识从现实向梦境转移

影子状态：从依附转为游离

唤醒切换

触发条件：梦主醒来

切换过程：意识从梦境返回现实

影子状态：从游离回归依附

双模式特性对比

维度 白天模式（现实） 夜晚模式（梦境）
时间感知 线性、稳定 非线性、可压缩/延展
空间结构 固定、三维 流动、多维
物理法则 牛顿力学 奇幻规则
成长类型 永久性 临时性
风险程度 低 高
情绪影响 积累 释放/具象化

梦境生成原理

世界构建机制

素材来源

1. 记忆碎片：日常经历的片段重组

2. 情绪残留：未处理的情绪能量

3. 欲望投射：潜意识愿望的象征表达

4. 外部影响：环境刺激（声音、温度等）

稳定性原理

稳定因素

梦主情绪平稳、熟悉的场景和人物、重复出现的主题

不稳定因素

强烈情绪波动、陌生环境、外部干扰（如闹钟）

📚 理论框架

梵天一梦世界观框架

核心隐喻

整个世界是某个存在的梦境，梦境中的存在也是其他存在的梦主，层层嵌套，互为表里

哲学基础：印度哲学中的"摩耶"（幻象）概念、庄子的"庄周梦蝶"思想、现代模拟理论

游戏体现：梦境世界的荒诞逻辑、现实与梦境的模糊边界、多重梦境的可能性

成长心理学框架

埃里克森心理社会发展阶段

学龄期（6-12岁）

主题：勤奋对自卑

游戏中的体现：通过战斗和探索建立能力感

青春期（12-18岁）

主题：自我同一性对角色混乱

游戏中的体现：影子与宿主的身份认同探索

荣格分析心理学

原型应用

阴影（Shadow）：被压抑的自我部分

阿尼玛/阿尼姆斯：异性特质

智者：引导者角色（魔女、医生）

母亲原型：魔女角色

叙事结构框架

英雄之旅（Hero's Journey）

阶段映射

  1. 普通世界：小男孩的日常生活
  2. 冒险召唤：影子觉醒，进入梦境
  3. 拒绝召唤：对梦境世界的恐惧
  4. 遇见导师：魔女、医生等引导者
  5. 跨越第一道门槛：首次深入梦境
  6. 考验、盟友、敌人：战斗与成长
  7. 深入洞穴：面对内心最深的恐惧
  8. 磨难：与最终敌人的决战
  9. 奖赏：获得成长和理解
  10. 归途：返回现实世界
  11. 复活：现实中的改变
  12. 携万能药归来：将梦境中的成长应用于现实

🔮 抽象机制

存在维持机制

存在公式

自我意识 = 光能 × 战斗意志 × 时间

消逝过程

  1. 初期：感觉迟钝，反应变慢
  2. 中期：记忆模糊，情感淡化
  3. 后期：自我认知混乱
  4. 末期：完全失去意识，沦为普通影子

维持方式

  • 光能补充：接触光源
  • 战斗刺激：战斗中的肾上腺素效应
  • 情感连接：与宿主或其他存在的羁绊

情绪-环境耦合机制

耦合原理

梦主的情绪状态直接影响梦境环境，环境变化反过来影响梦主的情绪，形成正反馈或负反馈循环

耦合实例

情绪状态 环境表现 环境影响
恐惧 黑暗、狭窄空间 加剧恐惧感
平静 开阔、明亮场景 强化平静感
愤怒 火焰、破碎地形 激化愤怒
悲伤 雨、灰色色调 加深悲伤

认知重构机制

定义

通过梦境体验改变对现实的认知

重构过程

  1. 暴露：在梦境中面对恐惧源
  2. 体验：以安全的方式经历威胁
  3. 理解：获得新的认知框架
  4. 迁移：将新认知应用于现实

治疗应用

  • 创伤后应激障碍（PTSD）的处理
  • 恐惧症的暴露疗法
  • 自我认知的调整

故事设定

叙事

🌍 世界观背景

世界起源

远古时代

永恒太阳：天上有一个特殊的太阳，持续照耀大地，没有黑夜

影子繁荣：影子一族在永恒阳光下建立了自己的文明

繁荣景象：影子一族发展出独特的社会、艺术和技术

太阳衰败

黑夜诞生：太阳开始衰败，黑夜逐渐出现

影子危机：落入黑夜的影子逐渐失去自我，沦为普通影子

适应进化：部分影子学会依附人类来维持存在

🌙 梦境世界地理与睡眠阶段

梦境世界并非单一的空间，而是随着睡眠深度变化的多层结构。每一层对应不同的睡眠阶段，具有独特的环境特征、事件类型和游戏规则。

睡眠阶段与梦境层级

睡眠阶段 梦境层级 环境特征 核心玩法
清醒期 表层梦境 现实与梦境的模糊边界，熟悉的场景开始扭曲 探索、收集情报、与NPC对话
快速眼动期 活跃梦境层 色彩鲜艳、逻辑荒诞、充满象征意义 高强度战斗、剧情推进、BOSS战
浅睡期 过渡层 场景不稳定，在多个记忆片段间切换 解谜、寻找路径、躲避危险
深睡期 核心梦境层 黑暗、压抑、接近潜意识深处 极限生存、资源管理、面对深层恐惧

睡眠阶段详解

1. 清醒期（Hypnagogic State）

环境特征

现实场景开始产生微妙的变化，墙壁可能渗出液体，家具漂浮或变形，光线不稳定，声音回响异常

特殊事件

入梦幻象：看到即将进入的梦境的预兆画面

现实回响：现实中听到的声音以扭曲形式传入梦境

边界行者：遇到徘徊在清醒与梦境之间的迷失灵魂

2. 快速眼动期（REM Sleep）

环境特征

梦境最为活跃和生动的阶段，色彩饱和度极高，超现实主义风格，物理法则完全失效

特殊事件

幻象洪流：大量情绪和记忆同时涌现，形成视觉风暴

飞行体验：玩家可以在此阶段获得飞行能力

变形遭遇：敌人和环境会突然改变形态

3. 浅睡期（Light Sleep）

环境特征

场景在多个记忆片段间不稳定切换，地面可能突然变成水面，光线柔和但方向不断变化

特殊事件

记忆闪回：随机触发过去的记忆片段，提供背景故事

路径迷宫：场景不断重组，需要找到稳定的"锚点"

半梦半醒：偶尔能看到现实世界的模糊投影

4. 深睡期（Deep Sleep）

环境特征

接近纯粹的黑暗，只有微弱的光源，空间感极度压缩，声音低沉、缓慢，时间感极度延长

特殊事件

深渊凝视：面对自己内心最深层的恐惧（强制BOSS战）

时间凝滞：某些区域时间几乎停止，玩家可以思考策略

潜意识低语：听到来自内心深处的声音，揭示隐藏真相

特殊睡眠状态

梦中梦（Dream Within a Dream）

机制说明

触发条件：在深层梦境中入睡、使用"重梦境药剂"、被特定敌人攻击

环境特征：梦境中的梦境，现实感更强反而更加诡异，时间流速极慢

游戏机制：双重身份控制、现实扭曲影响外层、逃脱挑战

清醒梦（Lucid Dream）

机制说明

触发条件：找到"觉醒符号"、完成现实检验动作、随机触发

特殊能力：梦境控制、飞行能力、时间操控、创造模式

限制风险：能量持续消耗、过度使用导致"梦境崩塌"

鬼压床（Sleep Paralysis）

机制说明

触发条件：凌晨3-4点进入游戏、连续多次死亡、被诅咒类敌人攻击

核心机制：完全定身、只能使用精神技能、恐惧激增、QTE系统

压床灵类型：黑影人、低语者、重压者、观察者

👥 角色背景故事

主角：小男孩与影子

小男孩（现实身份）

基本情况：爱幻想的小男孩，敏感而孤独

健康状况：患有慢性健康问题，需要定期看医生

心理状态：渴望情感连接，容易感到孤独

影子（梦境身份）

前身：影子一族的残余，在太阳衰败后失去大部分力量

觉醒契机：被魔女的泪水唤醒，获得独立意识

核心困境：在黑夜中必须不断战斗来维持自我意识

魔女

角色设定

存在形态：幽灵，有本体的灵体（戒指/玩偶/耳环）

外貌原型：梦主（小男孩）的妈妈——只有晚上会出现，不愿展示软弱

性格特点：端庄优雅，傲娇，把自己最柔软的地方都隐藏起来

核心动机

终极目标：找到自己生前的家

真实渴望：想知道父母对自己的真实想法

隐藏柔软：外表坚强，内心渴望被理解

寻找心脏的小熊

背景故事

小男孩在夜晚感到深深的孤独，蜷缩在床上用力拥抱着小小的泰迪熊。他渴望更多的接触，仿佛这样就能获得更多宽慰。当他聆听着泰迪熊的心跳声（扑通扑通……）时，空调的冷风让他打了个喷嚏，胸口离开了小熊，但他仍然听到了扑通扑通的声音……原来一直都是他自己的心跳。

医生

背景故事

小男孩有慢性的健康问题，需要定期看医生。每次和医生对话后，医生都会和父母进行严肃的谈话，之后父母会显得很伤心和担忧。小男孩逐渐形成了条件反射：医生的出现 = 不好的事情发生 = 父母伤心。

真实身份：梦境世界的"免疫系统管理员"，看似带来痛苦，实则在治愈和保护

📖 主要剧情线

故事一：魔女的谎言

剧情概述

一个关于母性、理解与和解的故事。魔女编造谎言引导主角摧毁"太阳"，真实动机是寻找生前的家、了解父母对自己的真实想法。

故事弧线

  1. 唤醒：魔女在黎明时分离世，泪水唤醒影子
  2. 羁绊：影子与魔女一起旅行，建立情感连接
  3. 真相：影子发现魔女真正想要摧毁的"太阳"是她自己的执念
  4. 告别：影子阻止魔女，在故乡阳光下与她告别

故事二：寻找心脏的小熊

剧情概述

一个关于自我接纳的故事。小熊一直在寻找一颗真正的心脏，相信这样就能获得完整的生命。最终揭示，心跳声一直都在小熊"体内"——这是小男孩自己的心跳。

故事弧线

  1. 相遇：影子在梦境中遇到寻找心脏的小熊
  2. 旅程：一起踏上寻找心脏的旅程，收集"心脏碎片"
  3. 发现：发现心跳声是小男孩的心跳
  4. 接纳：小熊接纳了自己的本质，完成自我接纳

故事三：逃离那个医生

剧情概述

一个关于打破条件反射、理解痛苦意义的故事。医生在梦境中表现为恐怖的存在，最终揭示他实际上是梦境的保护者。

故事弧线

  1. 恐惧：医生以恐怖形象出现，带来灾厄
  2. 追逐：医生似乎一直在追踪主角
  3. 揭示：医生揭示自己的真实身份——免疫系统管理员
  4. 和解：从恐惧变为理解和感激

游戏性设定

机制

😴 睡眠阶段系统

游戏核心循环围绕睡眠阶段展开。每次"夜晚"（游戏关卡）都会经历不同的睡眠阶段，每个阶段提供独特的游戏体验和挑战。

标准睡眠周期

周期流程

清醒期 (5分钟) → 浅睡期 (10分钟) → 深睡期 (10分钟) → REM期 (15分钟)

每个周期约30-45分钟，一晚通常有3-5个周期

各阶段游戏机制

清醒期 - 准备阶段

机制 说明 玩家操作
路线选择 选择本次梦境的探索方向 在地图界面选择入口
情报收集 与守门人对话获取提示 对话选择
入梦碎片 消耗碎片解锁新区域 资源管理

浅睡期 - 过渡阶段

机制 说明 影响
状态波动 属性在-20%到+20%间随机波动 战斗不稳定
场景重组 地图布局不断变化 需要寻找锚点
记忆闪回 触发剧情片段 提供背景故事

深睡期 - 生存挑战

机制 说明 风险/收益
恐惧累积 恐惧值持续上升 过高会触发恐慌状态
资源稀缺 回复道具掉落率-70% 必须精打细算
真实伤害 受到的伤害10%延续到下次梦境 长期影响

恐惧系统

恐惧值等级

0-30：正常状态

31-60：轻微恐慌（命中率-10%）

61-80：中度恐慌（命中率-20%，移动速度-15%）

81-100：重度恐慌（无法控制角色，随机移动）

REM期 - 能力爆发

机制 说明 数值
战斗加成 攻击力提升 +50%
技能加速 技能冷却时间减少 -50%
创造力爆发 解锁特殊"梦境技能" 临时技能槽
高风险 敌人强度和密度提升 +40%

梦境技能（REM期专属）

技能名称 效果 消耗

游戏类型：伪3D俯视角像素风格肉鸽冒险游戏

游戏引擎：Godot 4.x

项目特色亮点

1. 双重世界设定：表层是剑与魔法+科幻融合的梦境冒险世界；底层真相是爱幻想小男孩的梦，影子是共生种族

2. 现实-梦境联动：小男孩入睡后影子进入梦境战斗，怪物是现实经历/情绪的投射

3. 核心主题：内心成长与自我和解，通过梦境冒险反映现实心理蜕变

核心概念

设计理念

🌌 世界观概述

《影梦》(Silhouette Dream) 构建了一个独特的双重世界系统，将现实与梦境深度结合：

梦境世界

表象：剑与魔法+科幻融合的奇幻冒险世界

真相：小男孩梦境的心理投射，所有元素都是现实经历和情绪的具象化

特点：随机生成的梦境关卡，情绪化的敌人和BOSS，影子共生战斗系统

现实世界

设定：小男孩的日常校园和家庭生活

作用：影响梦境内容和难度，提供永久成长解锁

特点：温馨写实风格，日常互动影响梦境战斗

🔄 核心玩法特色

  • 梦境冒险：伪3D俯视角肉鸽战斗，随机生成梦境关卡，局内拾取武器/道具/技能
  • 现实叙事：小男孩日常互动（校园、家庭、情绪变化），影响梦境内容和难度
  • 影子共生：影子作为主角在梦境中的化身，能力随现实成长而进化
  • 情绪投射：怪物和BOSS是现实负面情绪（如霸凌）的具象化
  • 双结局系统：普通结局（完成冒险）+ 隐藏真结局（揭露世界真相）

🤖 AI开发流程风险

技术风险

  • AI生成质量不稳定：准备人工审核和编辑流程，建立质量检查标准
  • 需求变更频繁：采用敏捷方法，小批量迭代，保持文档的灵活性
  • 引擎选择困难：进行技术原型验证，对比实际效果而非理论参数

协作风险

  • 团队成员理解偏差：定期召开评审会议，确保认知对齐
  • 文档维护不及时：建立文档更新流程，指定维护责任人

🚨 Agent协调阶段风险

  • 需求变更：必须更新文档并通知所有相关Agent
  • 技术障碍：记录在问题跟踪系统，24小时内响应
  • 资源缺失：使用占位符继续开发，标记为待完成

feat: fix: chore: docs: style: refactor: test:

🔄 分支策略

分支类型 创建来源 合并目标 用途
feature/ develop develop 新功能开发
bugfix/ develop develop Bug修复
hotfix/ main main 和 develop 紧急生产环境修复

🔀 Pull Request流程

  1. 开发者在本地完成功能开发并推送到远程分支
  2. 创建Pull Request到develop分支
  3. 至少需要一名团队成员代码审查通过
  4. 通过CI/CD流水线测试
  5. 合并到目标分支

🏷️ 标签管理

  • 使用语义化版本标签（v1.0.0）标记发布版本
  • 每个正式发布版本创建标签

⚡ GitHub Actions

配置自动化测试、构建和部署流水线
