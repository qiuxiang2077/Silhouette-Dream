项目概述

基础信息

🎮 游戏基本信息

项目名称：《影梦》 (暂定标题)

游戏类型：伪3D俯视角像素风格肉鸽冒险游戏

世界观基底：「梵天一梦」式梦境逻辑，融合现实成长叙事与共生设定

核心循环：梦境冒险（局内随机成长）+ 现实成长（局外永久解锁）

美术风格：伪3D俯视角像素风，梦境奇幻 vs 现实写实

游戏引擎：Godot 4.x

开发状态：概念设计阶段

像素风格 肉鸽冒险 伪3D俯视角 Godot引擎 概念设计阶段

🎯 参考游戏

《元气骑士》、《饥荒》、《喵喵启示录》、《星露谷物语》、《杀戮尖塔》、《土豆兄弟》

项目特色亮点

1. 双重世界设定：表层是剑与魔法+科幻融合的梦境冒险世界；底层真相是爱幻想小男孩的梦，影子是共生种族

2. 现实-梦境联动：小男孩入睡后影子进入梦境战斗，怪物是现实经历/情绪的投射

3. 核心主题：内心成长与自我和解，通过梦境冒险反映现实心理蜕变

核心概念

设计理念

🌌 世界观概述

《影梦》构建了一个独特的双重世界系统，将现实与梦境深度结合：

梦境世界

表象：剑与魔法+科幻融合的奇幻冒险世界

真相：小男孩梦境的心理投射，所有元素都是现实经历和情绪的具象化

特点：随机生成的梦境关卡，情绪化的敌人和BOSS，影子共生战斗系统

现实世界

设定：小男孩的日常校园和家庭生活

作用：影响梦境内容和难度，提供永久成长解锁

特点：温馨写实风格，日常互动影响梦境战斗

🔄 核心玩法特色

  • 梦境冒险：伪3D俯视角肉鸽战斗，随机生成梦境关卡，局内拾取武器/道具/技能
  • 现实叙事：小男孩日常互动（校园、家庭、情绪变化），影响梦境内容和难度
  • 影子共生：影子作为主角在梦境中的化身，能力随现实成长而进化
  • 情绪投射：怪物和BOSS是现实负面情绪（如霸凌）的具象化
  • 双结局系统：普通结局（完成冒险）+ 隐藏真结局（揭露世界真相）

Agent A

程序编程需求分析师

角色定位：技术翻译专家，将游戏策划需求转化为精确、可执行的技术规格。

  • 制定核心系统需求
  • 产出程序需求文档

输出：程序编程需求文档.md、技术风险评估报告.md

Agent B

插件脚本开发工程师

角色定位：Godot引擎专家，基于程序需求文档实现高质量、可维护的游戏代码。

  • 引擎环境准备
  • 核心系统实现
  • 质量保证

输出：Godot插件脚本包/、测试套件/、API文档.md

Agent C

美术音频设计师

角色定位：创意资源专家，负责游戏所有视觉和听觉元素的创作。

  • 美术设计（梦境/现实风格）
  • 音频设计（BGM/音效）

输出：美术音频资源规格书.md、资源清单.csv

角色设定

关键角色

🧸 寻找心脏的熊 (The Heart-Seeking Bear)

核心概念

心理学基础：投射 (Projection) 防御机制

情感核心：孤独感、对安慰的渴望、自我欺骗与自我接纳

象征意义：将自身的情感需求投射到外部对象，最终发现真相在于自我接纳

背景故事

现实层面：小男孩在夜晚感到深深的孤独，蜷缩在床上用力拥抱着小小的泰迪熊。他渴望更多的接触，仿佛这样就能获得更多宽慰。当他聆听着泰迪熊的心跳声（扑通扑通……）时，空调的冷风让他打了个喷嚏，胸口离开了小熊，但他仍然听到了扑通扑通的声音……原来一直都是他自己的心跳。

梦境映射：在梦境世界中，这只泰迪熊化身为一个寻找心脏的神秘生物。它相信只要找到一颗真正的心脏，就能获得完整的生命和情感能力。实际上，这是小男孩将自己对情感连接和生命意义的渴望投射到了泰迪熊身上。

角色设定

  • 外貌：一个破旧的棕色泰迪熊，右眼纽扣脱落，身上有缝补痕迹，胸口处有一个空心的洞
  • 大小：约30厘米高，可以被小男孩轻松抱在怀中
  • 特殊特征：空洞的胸口会随着小男孩的情绪波动而发出微弱的光芒
  • 声音：温柔但带着一丝悲伤的童声，说话时带有回声效果

⚕️ 医生 (The Doctor)

核心概念

现实基础：医疗创伤的条件反射

叙事设计：假反派 (False Antagonist)

角色弧线：从恐惧对象到拯救者的转变

背景故事

现实层面：小男孩有慢性的健康问题，需要定期看医生。每次和医生对话后，医生都会和父母进行严肃的谈话，之后父母会显得很伤心和担忧。小男孩逐渐形成了条件反射：医生的出现 = 不好的事情发生 = 父母伤心。

梦境映射：在梦境世界中，"医生"是一个B格很高、类似敌方干部的角色。他穿着白色的长袍，戴着鸟嘴面具，手持手术刀和医疗仪器。每次出现都会带来灾厄：区域瘟疫、怪物变异、环境恶化等。

角色设定

  • 外貌：身高约190cm，瘦削但威严的身形，穿着沾有不明污渍的白色长袍，戴着传统的瘟疫医生鸟嘴面具
  • 气质：冷静、理性、疏离，说话带有回声和金属质感
  • 出场效果：伴随绿色雾气、消毒水气味和心跳监视器的声音
  • 真实身份：梦境世界的"免疫系统管理员"，实际上站在主角这边

🔗 角色间关联与主题融合

情感互补

熊：代表柔软的情感需求、自我欺骗、自我接纳

医生：代表硬性的治疗痛苦、条件反射、真相面对

共同主题：都是小男孩内心世界的投射，都是帮助成长的机制

主题总结

这两个角色共同体现了《影梦》的核心主题：

  1. 表象vs真相：医生看似恐怖实为帮助，熊寻求外部心脏实为内部接纳
  2. 痛苦与成长：医生的"治疗性创伤"，熊的"孤独到自我接纳"
  3. 投射与内省：都是小男孩内心世界的投射，最终引导向自我认知
  4. 情感完整：从分离（熊与心脏分离）到整合（医生与主角结盟）的旅程

🤖 AI开发流程风险

技术风险

  • AI生成质量不稳定：准备人工审核和编辑流程，建立质量检查标准
  • 需求变更频繁：采用敏捷方法，小批量迭代，保持文档的灵活性
  • 引擎选择困难：进行技术原型验证，对比实际效果而非理论参数

协作风险

  • 团队成员理解偏差：定期召开评审会议，确保认知对齐
  • 文档维护不及时：建立文档更新流程，指定维护责任人

🚨 Agent协调阶段风险

  • 需求变更：必须更新文档并通知所有相关Agent
  • 技术障碍：记录在问题跟踪系统，24小时内响应
  • 资源缺失：使用占位符继续开发，标记为待完成

成功标准

目标设定

🎯 短期目标（4周）

  • 可运行原型：玩家在随机生成的1个房间内移动、攻击、击败1种敌人
  • 核心手感验证：移动流畅、攻击反馈清晰、基础UI显示正常

📈 中期目标（8周）

  • 完整游戏循环：清空多个房间 → 击败小Boss → 获取新武器 → 进入下一区域
  • 内容基础：3-5种敌人、5-8种武器、局外成长系统雏形

🏆 长期目标（16周）

  • 可发布版本：稳定的60FPS、无崩溃性错误、完整的从开始到第一个大Boss的体验
