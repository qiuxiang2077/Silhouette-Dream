# 《影梦》(Silhouette Dream) 游戏性设定文档

---

## 概述

本文档定义《影梦》(Silhouette Dream) 游戏的核心玩法机制、系统规格和交互设计。游戏性设定将概念设定转化为可执行的游戏规则，确保玩家体验与设计目标一致。

---

## 一、睡眠阶段系统

### 1.1 系统概述

游戏核心循环围绕睡眠阶段展开。每次"夜晚"（游戏关卡）都会经历不同的睡眠阶段，每个阶段提供独特的游戏体验和挑战。

**核心设计理念**：
- 睡眠阶段不仅是背景设定，而是核心游戏机制
- 玩家需要适应不同阶段的特性，调整策略
- 阶段之间的过渡是游戏节奏的关键节点

---

### 1.2 睡眠阶段循环

#### 标准睡眠周期（游戏内约30-45分钟）

```
清醒期 → 浅睡期 → 深睡期 → 快速眼动期 → [循环或结束]
   ↓        ↓        ↓          ↓
 5分钟   10分钟   10分钟     15分钟
```

**周期特性**：
- 每个周期约30-45分钟（游戏内时间）
- 一晚通常有3-5个周期
- 随着周期推进，REM期逐渐延长
- 玩家可以在周期结束时选择"醒来"或"继续睡"

---

### 1.3 各阶段游戏机制

#### 清醒期（Hypnagogic State）

**阶段定位**：准备阶段 / 策略规划

**核心机制**：

| 机制 | 说明 | 玩家操作 |
|-----|------|---------|
| 路线选择 | 选择本次梦境的探索方向 | 在地图界面选择入口 |
| 情报收集 | 与守门人对话获取提示 | 对话选择 |
| 装备整备 | 携带道具进入梦境 | 背包管理 |
| 入梦碎片 | 消耗碎片解锁新区域 | 资源管理 |

**特殊系统**：

- **入梦碎片系统**
  - 通过完成梦境目标获得
  - 用于解锁新的梦境入口
  - 不同类型碎片对应不同区域

- **守门人对话**
  - 每晚随机提供1-3条提示
  - 提示内容包括：敌人分布、宝藏位置、危险预警
  - 可以通过道具获得更多提示

**阶段目标**：
- 规划本次梦境的路线
- 根据提示调整装备和策略
- 为即将到来的挑战做准备

---

#### 浅睡期（Light Sleep）

**阶段定位**：过渡阶段 / 探索解谜

**核心机制**：

| 机制 | 说明 | 影响 |
|-----|------|------|
| 状态波动 | 属性在-20%到+20%间随机波动 | 战斗不稳定 |
| 场景重组 | 地图布局不断变化 | 需要寻找锚点 |
| 记忆闪回 | 触发剧情片段 | 提供背景故事 |
| 现实投影 | 偶尔看到现实世界的模糊影像 | 提示和干扰并存 |

**锚点系统**：
- 场景中存在的稳定点，不受重组影响
- 玩家需要找到锚点来"固定"当前区域
- 锚点类型：
  - **记忆锚点**：与特定记忆相关的物品
  - **情感锚点**：强烈情绪留下的痕迹
  - **意识锚点**：玩家主动创造的标记

**解谜要素**：
- 环境谜题为主，战斗为辅
- 谜题类型：
  - 路径选择：在重组的迷宫找到正确路线
  - 记忆拼图：重组破碎的记忆片段
  - 现实对照：利用现实投影找到隐藏通道

**阶段目标**：
- 穿越不稳定的区域
- 收集记忆碎片了解剧情
- 寻找通往下一阶段的锚点

---

#### 深睡期（Deep Sleep）

**阶段定位**：生存挑战 / 资源管理

**核心机制**：

| 机制 | 说明 | 风险/收益 |
|-----|------|----------|
| 恐惧累积 | 恐惧值持续上升 | 过高会触发恐慌状态 |
| 资源稀缺 | 回复道具掉落率-70% | 必须精打细算 |
| 真实伤害 | 受到的伤害10%延续到下次梦境 | 长期影响 |
| 安全点 | 唯一可以休息恢复的地方 | 位置隐蔽 |

**恐惧系统**：

```
恐惧值范围：0-100

0-30：正常状态
31-60：轻微恐慌（命中率-10%）
61-80：中度恐慌（命中率-20%，移动速度-15%）
81-100：重度恐慌（无法控制角色，随机移动）
```

**降低恐惧的方法**：
- 到达安全点：-30恐惧值
- 击败敌人：-5恐惧值
- 使用特定道具：-10到-20恐惧值
- 完成小目标：-15恐惧值

**安全点机制**：
- 每区域1-2个，位置随机
- 在安全点可以：
  - 完全恢复生命值
  - 降低恐惧值
  - 保存进度（检查点）
  - 整理背包

**阶段目标**：
- 在资源极度稀缺的情况下生存
- 管理恐惧值避免失控
- 找到通往REM期的路径

---

#### 快速眼动期（REM Sleep）

**阶段定位**：高潮战斗 / 能力爆发

**核心机制**：

| 机制 | 说明 | 数值 |
|-----|------|------|
| 战斗加成 | 攻击力提升 | +50% |
| 技能加速 | 技能冷却时间减少 | -50% |
| 创造力爆发 | 解锁特殊"梦境技能" | 临时技能槽 |
| 高风险 | 敌人强度和密度提升 | +40% |

**梦境技能系统**：

REM期专属技能，只能在当前阶段使用：

| 技能名称 | 效果 | 消耗 |
|---------|------|------|
| 现实扭曲 | 改变局部地形 | 梦境能量×30 |
| 时间裂隙 | 暂停时间3秒 | 梦境能量×50 |
| 幻象分身 | 创造2个分身吸引敌人 | 梦境能量×40 |
| 情绪爆发 | 释放储存的情绪造成范围伤害 | 当前恐惧值×2 |
| 记忆具现 | 召唤记忆中的强力武器 | 梦境能量×60 |

**梦境能量获取**：
- 击败敌人：+10-30能量
- 破坏环境：+5能量
- 完成挑战：+50能量
- 能量不会延续到下一阶段

**BOSS战机制**：
- 每个REM期结束时强制触发
- BOSS类型与当前情绪主题相关
- 击败BOSS获得丰厚奖励和剧情推进

**阶段目标**：
- 利用能力加成快速推进
- 击败REM期BOSS
- 最大化收集稀有资源

---

### 1.4 特殊睡眠状态机制

#### 梦中梦（Dream Within a Dream）

**触发概率**：5-10%（可通过道具提升）

**核心机制**：

| 机制 | 说明 |
|-----|------|
| 双重身份 | 同时控制影子和梦中的影子 |
| 时间流速 | 外层1分钟 = 内层1小时 |
| 现实扭曲 | 内层改变影响外层 |
| 逃脱挑战 | 必须找到"醒来"方法 |

**梦中梦层级**：
- 最多3层嵌套
- 每深入一层，现实感越强，诡异感越重
- 深层的时间流速更快

**逃脱方法**：
1. 找到"觉醒符号"（几何图案）
2. 完成"现实检验"（如照镜子、阅读文字）
3. 被外层敌人攻击（强制唤醒，有伤害）

**奖励机制**：
- 在梦中梦中获得的经验和道具会带到外层
- 完成深层梦中梦解锁特殊成就

---

#### 清醒梦（Lucid Dream）

**触发方式**：
- 被动：找到觉醒符号（20%概率）
- 主动：后期解锁技能主动触发（消耗资源）

**核心机制**：

| 能力 | 效果 | 消耗/限制 |
|-----|------|----------|
| 环境控制 | 召唤/改变地形 | 梦境能量 |
| 飞行 | 无限制飞行 | 无消耗，但吸引敌人 |
| 时间操控 | 暂停/缓慢时间 | 大量梦境能量 |
| 创造 | 生成临时武器/道具 | 梦境能量+素材 |

**梦境能量管理**：
- 清醒梦状态下持续消耗能量
- 初始能量：100点
- 自然消耗：-5点/秒
- 使用能力：额外消耗

**崩塌风险**：
- 能量耗尽或过度使用能力会触发"梦境崩塌"
- 崩塌效果：
  - 强制退出清醒梦状态
  - 受到最大生命值30%的伤害
  - 恐惧值+50
  - 当前区域敌人全部刷新

**阶段目标**：
- 利用能力探索隐藏区域
- 收集平时无法获得的稀有物品
- 在能量耗尽前完成目标

---

#### 鬼压床（Sleep Paralysis）

**触发条件**：
- 时间：凌晨3-4点进入游戏（现实时间彩蛋）
- 死亡：连续3次在战斗中死亡
- 诅咒：被特定敌人攻击

**核心机制**：

| 机制 | 说明 |
|-----|------|
| 完全定身 | 无法移动、攻击、使用道具 |
| 精神技能 | 只能使用特殊的"精神技能" |
| 恐惧激增 | 恐惧值快速上升 |
| QTE系统 | 通过节奏游戏保持理智 |

**精神技能**：

| 技能 | 效果 | 冷却 |
|-----|------|------|
| 意志抵抗 | 降低恐惧值增长速度 | 10秒 |
| 精神护盾 | 抵挡一次压床灵攻击 | 15秒 |
| 呼吸节奏 | 恢复少量理智值 | 5秒 |
| 光明想象 | 暂时驱散周围黑暗 | 20秒 |

**压床灵战斗**：
- 4种类型压床灵随机出现
- 每种有独特的攻击模式和应对策略
- 必须通过精神技能对抗

**逃脱机制**：
1. **自然结束**：持续60-120秒后自动结束
2. **QTE成功**：完成节奏小游戏提前结束
3. **外部干扰**：现实中被"唤醒"（随机事件）

**奖励机制**：
- 成功抵抗：获得"意志"属性永久+1
- 完美QTE：额外获得特殊道具
- 多次成功：解锁"梦魇征服者"成就

---

### 1.5 阶段转换与进度系统

#### 阶段转换事件

**自然转换**：
- 完成当前阶段目标后自动进入下一阶段
- 有过渡动画和提示

**强制转换**：
- 在某一阶段停留过久会被强制推进
- 通常是因为"睡眠周期"的自然推进

**意外转换**：
- 特定事件可以跳跃阶段
- 如：在深睡期找到"REM之门"可直接进入REM期

#### 睡眠周期进度

**周期计数器**：
- UI显示当前是第几个周期
- 每个周期的REM期逐渐延长
- 第3个周期后解锁"深层梦境"区域

**周期奖励**：
- 完成完整周期获得"睡眠质量"评分
- 高评分获得额外奖励：
  - 经验值加成
  - 稀有掉落率提升
  - 解锁新剧情片段

## 四、UI/UX设计

### 4.1 睡眠阶段UI

**阶段指示器**：
- 屏幕边缘显示当前睡眠阶段
- 用颜色和图标区分不同阶段
- 显示距离下一阶段的时间/进度

**恐惧值显示**：
- 深睡期专用UI元素
- 心跳动画随恐惧值加速
- 屏幕边缘红色滤镜随恐惧值加深

**梦境能量条**：
- REM期和清醒梦专用
- 显示当前能量值和消耗速度
- 低能量时闪烁警告

### 4.2 特殊状态UI

**梦中梦指示**：
- 屏幕四角显示当前嵌套层数
- 每层有不同的视觉滤镜效果

**鬼压床UI**：
- 屏幕固定，无法转动视角
- QTE提示在屏幕中央
- 压床灵在视野边缘出现
