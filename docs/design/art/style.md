> **版本**: 1.0  
> **最后更新**: 2026年4月12日

---

## 一、整体美术风格

### 1.1 风格定位

**千禧年梦核风 (Chinese Millennium Dreamcore)**

以改革开放后中国快速城市化时期的后现代主义建筑为载体，营造令00后产生强烈怀旧感的梦境世界。

**核心氛围**: "你可以回到这里，但这里已经没有人了"

### 1.2 视觉关键词

- **建筑特征**: 蓝色/绿色玻璃、白色瓷砖贴片外立面、几何形外立面、旋转楼梯
- **时间痕迹**: 人去楼空、破旧不堪、玻璃碎片、厚厚灰尘
- **梦核元素**: 童年物品、发白的家具、褪色的装饰、未完成的日常
- **情绪基调**: 怀旧、孤独、神秘、超现实

### 1.3 参考时期

**主要参考**: 1990年代-2000年代中国城市化建筑
- 后现代主义商业建筑
- 国营商场、百货大楼
- 单位大院、职工宿舍
- 学校、医院、图书馆

## 三、场景设定

### 3.1 核心场景列表

#### A. 废弃商场
**核心元素**:
- 玻璃天幕（长满绿色/黑色苔藓）
- 悬挂的装饰球（未取下，褪色的节日装饰）
- 倒塌的货架（商品散落一地）
- 玻璃碎片（地面反射）
- 厚厚的灰尘（一切都被覆盖）
- 来不及搬走的商品（过时的商品包装）
- 纸质脆烂的宣传单（泛黄的促销海报）
- 假人模特（服装店的塑料模特，姿势诡异）
- 胶囊电梯（圆形观光电梯，停止运转）
- 发白的皮沙发（候客区的旧沙发）
- 螺旋楼梯（连接各楼层）

**氛围描述**: 
曾经繁华的商业中心，如今人去楼空。阳光透过长满苔藓的玻璃天幕洒下，照亮漂浮的灰尘。悬挂的装饰球在风中微微摇晃，发出轻微的碰撞声。

#### B. 废弃学校
**核心元素**:
- 教室（掉漆的课桌椅、发白的黑板）
- 宣传栏（泛黄的纸张、褪色的奖状）
- 操场（生锈的篮球架、开裂的水泥地）
- 走廊（墙上的涂鸦、破碎的窗户）

**墙上涂鸦**: "让所有理智都埋葬在____深处"

#### C. 废弃游乐场
**核心元素**:
- 旋转木马（褪色的木马、停止的音乐）
- 滑梯（生锈的金属、杂草丛生）
- 秋千（铁链生锈、座椅破损）
- 碰碰车场地（废弃的车辆、剥落的油漆）

#### D. 废弃小诊所
**核心元素**:
- 候诊区（发白的皮沙发、旧杂志）
- 药柜（玻璃破碎、药品散落）
- 诊疗室（生锈的器械、泛白的床单）
- 消毒水味道的视觉化（冷色调、反光地面）

#### E. 废弃图书馆/大剧院
**核心元素**:
- 书架（倒塌的书架、散落的书籍）
- 阅览区（积灰的桌椅、破损的台灯）
- 舞台（褪色的幕布、生锈的灯光设备）
- 观众席（破损的座椅、厚厚的灰尘）

### 3.2 场景通用元素

**建筑外立面**:
- 蓝色/绿色玻璃幕墙
- 白色瓷砖贴片
- 几何形外立面设计
- 旋转楼梯（外露式）

**时间痕迹**:
- 裂缝（墙面、地面的裂缝）
- 锈迹（金属构件的锈蚀）
- 蜘蛛网（角落的蛛网）
- 水渍（墙面上的渗水痕迹）

**植物入侵**:
- 从缝隙长出的杂草
- 爬满墙壁的藤蔓
- 地面裂缝中的野草
- 苔藓（潮湿处的绿色覆盖）

## 四、角色设定

### 4.1 主角：小男孩（现实）

**基础信息**:
- 尺寸: 32x32 像素
- 风格: 像素画，千禧年梦核风格

**外观设定**:
- 年龄: 约10-12岁
- 穿着: 校服或简单的T恤短裤（2000年代风格）
- 发型: 简单的学生头
- 表情: 敏感、孤独、略带忧郁
- 随身物品: 旧书包、泰迪熊（可选）

**动画需求**:
| 动画 | 帧数 | 描述 |
|-----|------|------|
| 待机(Idle) | 4帧 | 轻微呼吸，偶尔张望 |
| 走路(Walk) | 8帧 | 正常步行，略带沉重感 |
| 表情差分 | - | 正常、悲伤、惊讶、害怕 |

### 4.2 主角：影子（梦境）

**基础信息**:
- 尺寸: 32x32 像素
- 风格: 纯黑剪影，边缘发光

**外观设定**:
- 形态: 人形黑影，轮廓随光线波动
- 边缘: 轻微的发光边缘（代表与光的联系）
- 动态: 非实体感，流动性的移动
- 表情: 通过身体轮廓变化表现情绪

**特殊效果**:
- 在黑暗中会逐渐变得模糊（失去清晰度）
- 靠近光源时边缘更明亮
- 战斗时会有波动效果

**动画需求**:
| 动画 | 帧数 | 描述 |
|-----|------|------|
| 待机(Idle) | 4帧 | 黑影波动，呼吸感 |
| 走路(Walk) | 8帧 | 流动感移动，无实体 footsteps |
| 受击 | 4帧 | 闪烁、变形 |
| 技能释放 | 6帧 | 影子扩散、聚集 |

### 4.3 NPC：魔女

**基础信息**:
- 尺寸: 48x48 像素（比主角大）
- 风格: 幽灵形态，半透明

**外观设定**:
- 形态: 女性幽灵，半透明
- 服装: 端庄的长裙（类似母亲的形象）
- 特征: 优雅但孤独的气质
- 本体: 洋娃娃/戒指/耳环（需要单独设计）

**动画需求**:
| 动画 | 帧数 | 描述 |
|-----|------|------|
| 漂浮待机 | 6帧 | 幽灵般的上下漂浮 |
| 说话 | 4帧 | 手势配合 |
| 情绪波动 | - | 从优雅到悲伤的变化 |

### 4.4 NPC：寻找心脏的小熊

**基础信息**:
- 尺寸: 32x32 像素
- 风格: 破旧的泰迪熊

**外观设定**:
- 形态: 棕色泰迪熊，约30厘米高
- 破损: 右眼纽扣脱落、身上有缝补痕迹
- 核心特征: 胸口有一个空心的洞
- 特效: 空洞胸口有微弱的光芒（情感粒子）

**动画需求**:
| 动画 | 帧数 | 描述 |
|-----|------|------|
| 待机 | 4帧 | 抱胸姿势，轻微摇摆 |
| 走路 | 8帧 | 蹒跚步伐 |
| 倾听 | - | 专注倾听心跳的姿态 |

### 4.5 敌人：医生（梦境形态）

**基础信息**:
- 尺寸: 48x48 像素
- 风格: 瘟疫医生 + 医疗恐怖

**外观设定**:
- 身高: 约190cm（高大威严）
- 服装: 沾有污渍的白色长袍
- 面具: 传统瘟疫医生鸟嘴面具，深红色镜片
- 道具: 手术刀、医疗日志
- 特效: 伴随绿色雾气、消毒水气味视觉化

**动画需求**:
| 动画 | 帧数 | 描述 |
|-----|------|------|
| 待机 | 4帧 | 冷静的医疗姿势 |
| 走路 | 8帧 | 沉稳步伐 |
| 攻击 | 6帧 | 手术刀挥舞 |
| 治愈形态 | - | 面具下温和面容的揭示 |

## 六、场景Tileset规范

### 6.1 像素规格

- **基础单位**: 32x32 像素
- **地板/墙面**: 32x32 或 16x16
- **大型物体**: 64x64 或更大
- **小道具**: 16x16

### 6.2 Tileset分类

#### A. 建筑外立面
- 蓝色玻璃墙（完整、破损、带苔藓版本）
- 白色瓷砖墙（完整、开裂、脱落版本）
- 几何装饰块（后现代主义建筑特征）
- 窗框（金属、生锈版本）

#### B. 地板
- 水磨石地板（完整、积灰、碎裂版本）
- 瓷砖地板（完整、破损、污渍版本）
- 水泥地（开裂、杂草版本）
- 地毯（发白、破损版本）

#### C. 墙面
- 白墙（干净、污渍、涂鸦版本）
- 瓷砖墙（厨房/卫生间风格）
- 黑板（发白、字迹模糊）
- 宣传栏（泛黄、破损）

#### D. 装饰物
- 墙角线（木质、脱落）
- 管道（生锈、漏水）
- 灯具（破损、闪烁）
- 通风口（生锈、堵塞）

## 七、特效与氛围

### 7.1 光影效果

- **透过玻璃的光**: 冷色调的蓝色/绿色光束
- **灰尘粒子**: 阳光下的漂浮灰尘
- **闪烁灯光**: 故障的荧光灯效果
- **阴影**: 长阴影营造孤独感

### 7.2 环境特效

- **雾气**: 低处的薄雾
- **孢子/花粉**: 漂浮的微粒
- **萤火虫/光点**: 梦境中的神秘光点
- **情绪粒子**: 与情绪相关的颜色粒子

### 7.3 UI风格

- **主菜单**: 老旧电视机雪花效果过渡
- **对话框**: 类似旧照片的风格，泛黄边缘
- **物品栏**: 旧书包/背包的视觉呈现
- **地图**: 手绘风格的地图，有咖啡渍痕迹
