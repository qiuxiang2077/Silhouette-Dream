# 《影梦》(Silhouette Dream) 游戏性设定文档

---

## 概述

本文档定义《影梦》(Silhouette Dream) 游戏的核心玩法机制、系统规格和交互设计。游戏性设定将概念设定转化为可执行的游戏规则，确保玩家体验与设计目标一致。

---

## 零、开场与游戏流程

### 0.1 开场设计

#### 现实场景：黄昏归家

**时间节点**：黄昏时分

**场景描述**：
- 画面呈现一个神情疲惫的小男孩，背着书包走在回家的路上
- 天空被夕阳染成暖橙色，但光线逐渐暗淡
- 街道安静，偶尔传来远处的车声和风声

**场景过渡**：
1. **归家之路**：小男孩穿过熟悉的街道，步伐沉重
2. **抵达家门**：推开略显破旧的门，进入家中
3. **瘫倒床上**：卸下书包，直接扑倒在床上，一动不动
4. **开始游戏**：屏幕中央浮现「点击开始游戏」的提示

**点击后的演出**：
- 小男孩闭上眼睛，呼吸逐渐平稳
- 画面逐渐变暗，进入梦境过渡
- 影子从床边的黑暗中缓缓浮现
- 影子钻入小男孩的梦境之中
- 正式过渡到游戏主界面

#### 现实部分的极简设计

**核心理念**：现实部分仅作为引子，梦境才是游戏的主体

**现实场景元素**：
| 元素 | 描述 | 象征意义 |
|-----|------|---------|
| 神情疲惫的小男孩 | 略显病态的苍白面容，眼神中带着与年龄不符的忧郁 | 暗示健康状况与心理状态 |
| 破旧的家 | 墙壁斑驳，家具陈旧，但收拾得整洁 | 物质匮乏但精神富足 |
| 床 | 略显狭小的单人床，铺着洗得发白的床单 | 安全的港湾，梦境的入口 |

**家的氛围**：虽然破破烂烂，但是温馨
- 墙上贴着小男孩的画作
- 桌上摆放着简单的晚餐（可能是父母留下的）
- 床头放着一只旧泰迪熊

> 【来自团队讨论】现实部分就只有这么一点点——一个神情疲惫的小男孩、一个环境破旧的家、一张床。这种极简的现实设计让玩家快速进入梦境核心体验。

## 一、睡眠阶段系统

### 1.1 系统概述

游戏核心循环围绕梦境展开。每次"夜晚"（游戏关卡）都会经历不同的睡眠阶段，每个阶段提供独特的游戏体验和挑战。

**核心设计理念**：
- 梦境不仅是背景设定，而是核心游戏机制
- 玩家需要适应不同阶段的特性，调整策略
- 阶段之间的过渡是游戏节奏的关键节点

---

### 1.2 睡眠阶段循环

#### 标准睡眠周期（游戏内约30-45分钟）

```
清醒期 → 浅睡期 → 深睡期 → 快速眼动期 → [循环或结束]
   ↓        ↓        ↓          ↓
 5分钟   10分钟   10分钟     15分钟
```

**周期特性**：
- 每个周期约30-45分钟（游戏内时间）
- 一晚通常有3-5个周期
- 随着周期推进，REM期逐渐延长
- 玩家可以在周期结束时选择"醒来"或"继续睡"

---

### 1.3 各阶段游戏机制

#### 清醒期（Hypnagogic State）

**阶段定位**：准备阶段 / 策略规划

**核心机制**：

| 机制 | 说明 | 玩家操作 |
|-----|------|---------|
| 路线选择 | 选择本次梦境的探索方向 | 在地图界面选择入口 |
| 情报收集 | 与守门人对话获取提示 | 对话选择 |
| 装备整备 | 携带道具进入梦境 | 背包管理 |
| 入梦碎片 | 消耗碎片解锁新区域 | 资源管理 |

**特殊系统**：

- **入梦碎片系统**
  - 通过完成梦境目标获得
  - 用于解锁新的梦境入口
  - 不同类型碎片对应不同区域

- **守门人对话**
  - 每晚随机提供1-3条提示
  - 提示内容包括：敌人分布、宝藏位置、危险预警
  - 可以通过道具获得更多提示

**阶段目标**：
- 规划本次梦境的路线
- 根据提示调整装备和策略
- 为即将到来的挑战做准备

---

#### 浅睡期（Light Sleep）

**阶段定位**：过渡阶段 / 探索解谜

**核心机制**：

| 机制 | 说明 | 影响 |
|-----|------|------|
| 状态波动 | 属性在-20%到+20%间随机波动 | 行动不稳定 |
| 场景重组 | 地图布局不断变化 | 需要寻找锚点 |
| 记忆闪回 | 触发剧情片段 | 提供背景故事 |
| 现实投影 | 偶尔看到现实世界的模糊影像 | 提示和干扰并存 |

**锚点系统**：
- 场景中存在的稳定点，不受重组影响
- 玩家需要找到锚点来"固定"当前区域
- 锚点类型：
  - **记忆锚点**：与特定记忆相关的物品
  - **情感锚点**：强烈情绪留下的痕迹
  - **意识锚点**：玩家主动创造的标记

**解谜要素**：
- 环境谜题为主，战斗为辅
- 谜题类型：
  - 路径选择：在重组的迷宫找到正确路线
  - 记忆拼图：重组破碎的记忆片段
  - 现实对照：利用现实投影找到隐藏通道

**阶段目标**：
- 穿越不稳定的区域
- 收集记忆碎片了解剧情
- 寻找通往下一阶段的锚点

---

#### 深睡期（Deep Sleep）

**阶段定位**：生存挑战 / 资源管理

**核心机制**：

| 机制 | 说明 | 风险/收益 |
|-----|------|----------|
| 恐惧累积 | 恐惧值持续上升 | 过高会触发恐慌状态 |
| 资源稀缺 | 回复道具掉落率-70% | 必须精打细算 |
| 真实伤害 | 受到的伤害10%延续到下次梦境 | 长期影响 |
| 安全点 | 唯一可以休息恢复的地方 | 位置隐蔽 |

**恐惧系统**：

```
恐惧值范围：0-100

0-30：正常状态
31-60：轻微恐慌（命中率-10%）
61-80：中度恐慌（命中率-20%，移动速度-15%）
81-100：重度恐慌（无法控制角色，随机移动）
```

**降低恐惧的方法**：
- 到达安全点：-30恐惧值
- 击败敌人：-5恐惧值
- 使用特定道具：-10到-20恐惧值
- 完成小目标：-15恐惧值

**安全点机制**：
- 每区域1-2个，位置随机
- 在安全点可以：
  - 完全恢复生命值
  - 降低恐惧值
  - 保存进度（检查点）
  - 整理背包

**阶段目标**：
- 在资源极度稀缺的情况下生存
- 管理恐惧值避免失控
- 找到通往REM期的路径

---

#### 快速眼动期（REM Sleep）

**阶段定位**：高潮战斗 / 能力爆发

**核心机制**：

| 机制 | 说明 | 数值 |
|-----|------|------|
| 战斗加成 | 攻击力提升 | +50% |
| 技能加速 | 技能冷却时间减少 | -50% |
| 创造力爆发 | 解锁特殊"梦境技能" | 临时技能槽 |
| 高风险 | 敌人强度和密度提升 | +40% |

**梦境技能系统**：

REM期专属技能，只能在当前阶段使用：

| 技能名称 | 效果 | 消耗 |
|---------|------|------|
| 现实扭曲 | 改变局部地形 | 梦境能量×30 |
| 时间裂隙 | 暂停时间3秒 | 梦境能量×50 |
| 幻象分身 | 创造2个分身吸引敌人 | 梦境能量×40 |
| 情绪爆发 | 释放储存的情绪造成范围伤害 | 当前恐惧值×2 |
| 记忆具现 | 召唤记忆中的强力能力 | 梦境能量×60 |

**梦境能量获取**：
- 击败敌人：+10-30能量
- 破坏环境：+5能量
- 完成挑战：+50能量
- 能量不会延续到下一阶段

**关键挑战机制**：
- 每个REM期结束时触发关键剧情挑战
- 挑战类型与当前情绪主题相关
- 完成挑战获得丰厚奖励和剧情推进

**阶段目标**：
- 利用能力加成快速推进
- 击败REM期BOSS
- 最大化收集稀有资源

---

### 1.4 特殊睡眠状态机制

#### 梦中梦（Dream Within a Dream）

**触发概率**：5-10%（可通过道具提升）

**核心机制**：

| 机制 | 说明 |
|-----|------|
| 双重身份 | 同时控制影子和梦中的影子 |
| 时间流速 | 外层1分钟 = 内层1小时 |
| 现实扭曲 | 内层改变影响外层 |
| 逃脱挑战 | 必须找到"醒来"方法 |

**梦中梦层级**：
- 最多3层嵌套
- 每深入一层，现实感越强，诡异感越重
- 深层的时间流速更快

**逃脱方法**：
1. 找到"觉醒符号"（几何图案）
2. 完成"现实检验"（如照镜子、阅读文字）
3. 被外层敌人发现（强制唤醒，有惊吓值）

**奖励机制**：
- 在梦中梦中获得的经验和道具会带到外层
- 完成深层梦中梦解锁特殊成就

---

#### 清醒梦（Lucid Dream）

**触发方式**：
- 被动：找到觉醒符号（20%概率）
- 主动：后期解锁技能主动触发（消耗资源）

**核心机制**：

| 能力 | 效果 | 消耗/限制 |
|-----|------|----------|
| 环境控制 | 召唤/改变地形 | 梦境能量 |
| 飞行 | 无限制飞行 | 无消耗，但吸引敌人 |
| 时间操控 | 暂停/缓慢时间 | 大量梦境能量 |
| 创造 | 生成临时武器/道具 | 梦境能量+素材 |

**梦境能量管理**：
- 清醒梦状态下持续消耗能量
- 初始能量：100点
- 自然消耗：-5点/秒
- 使用能力：额外消耗

**崩塌风险**：
- 能量耗尽或过度使用能力会触发"梦境崩塌"
- 崩塌效果：
  - 强制退出清醒梦状态
  - 受到最大生命值30%的伤害
  - 恐惧值+50
  - 当前区域敌人全部刷新

**阶段目标**：
- 利用能力探索隐藏区域
- 收集平时无法获得的稀有物品
- 在能量耗尽前完成目标

---

#### 鬼压床（Sleep Paralysis）

**触发条件**：
- 时间：凌晨3-4点进入游戏（现实时间彩蛋）
- 死亡：连续3次在战斗中死亡
- 诅咒：被特定敌人攻击

**核心机制**：

| 机制 | 说明 |
|-----|------|
| 完全定身 | 无法移动、互动、使用道具 |
| 精神技能 | 只能使用特殊的"精神技能" |
| 恐惧激增 | 恐惧值快速上升 |
| QTE系统 | 通过节奏游戏保持理智 |

**精神技能**：

| 技能 | 效果 | 冷却 |
|-----|------|------|
| 意志抵抗 | 降低恐惧值增长速度 | 10秒 |
| 精神护盾 | 抵挡一次压床灵侵袭 | 15秒 |
| 呼吸节奏 | 恢复少量理智值 | 5秒 |
| 光明想象 | 暂时驱散周围黑暗 | 20秒 |

**压床灵对抗**：
- 4种类型压床灵随机出现
- 每种有独特的侵袭模式和应对策略
- 必须通过精神技能抵抗

**逃脱机制**：
1. **自然结束**：持续60-120秒后自动结束
2. **QTE成功**：完成节奏小游戏提前结束
3. **外部干扰**：现实中被"唤醒"（随机事件）

**奖励机制**：
- 成功抵抗：获得"意志"属性永久+1
- 完美QTE：额外获得特殊道具
- 多次成功：解锁"梦魇征服者"成就

---

### 1.5 阶段转换与进度系统

#### 阶段转换事件

**自然转换**：
- 完成当前阶段目标后自动进入下一阶段
- 有过渡动画和提示

**强制转换**：
- 在某一阶段停留过久会被强制推进
- 通常是因为"睡眠周期"的自然推进

**意外转换**：
- 特定事件可以跳跃阶段
- 如：在深睡期找到"REM之门"可直接进入REM期

#### 睡眠周期进度

**周期计数器**：
- UI显示当前是第几个周期
- 每个周期的REM期逐渐延长
- 第3个周期后解锁"深层梦境"区域

**周期奖励**：
- 完成完整周期获得"睡眠质量"评分
- 高评分获得额外奖励：
  - 经验值加成
  - 稀有掉落率提升
  - 解锁新剧情片段

## 二、互动与解谜系统

> 【来自团队讨论】游戏去掉了战斗环节，保留追逐、躲避等紧张元素，重点放在剧情和情感体验上，可能添加解谜、恐怖（一点点，不是主要的）元素。参考游戏：《小小梦魇》《Ender Lilies》《空洞骑士》《绣湖》《寻找天堂》等。

### 2.1 基础互动机制

**核心互动类型**：

| 互动类型 | 说明 | 典型场景 |
|---------|------|---------|
| 环境互动 | 与场景中的物件交互 | 拉开关、推箱子、捡取物品 |
| 潜行躲避 | 在敌人视野外行动 | 躲在障碍物后、等待敌人巡逻 |
| 追逐逃脱 | 被追踪时的逃跑环节 | 快速反应、路线选择 |
| 机关解谜 | 利用环境机制解决谜题 | 踏板序列、记忆游戏、路径规划 |

**移动与操作**：

| 操作 | 基础功能 | 特殊功能（梦境能力） |
|-----|---------|-----------------|
| 移动 | 方向键/WASD控制 | 二段跳、滑翔、穿墙 |
| 互动 | E键/空格键触发 | 远距离互动、范围互动 |
| 躲避 | 闪避翻滚 | 瞬移、隐身 |
| 观察 | 环顾四周 | 透视、标记目标 |

### 2.2 追逐与躲避机制

**追逐系统设计**：

| 机制 | 说明 | 设计参考 |
|-----|------|---------|
| 追踪AI | 敌人根据声音、视线追踪玩家 | 《小小梦魇》追逐设计 |
| 视野锥 | 敌人有可视范围和盲区 | 潜行游戏标准设计 |
| 警报状态 | 敌人发现玩家后进入追逐 | 追逐计时、距离衰减 |
| 逃脱判定 | 到达安全点或甩开追踪后逃脱 | 逃脱QTE、路线选择 |

**躲避策略**：

- **掩体系统**：利用环境物件躲避敌人视线
- **声东击西**：制造声响分散敌人注意力
- **路线规划**：观察敌人巡逻模式，选择安全路径
- **时机把握**：在敌人转身或分心的瞬间行动

**紧张度设计**：

| 元素 | 低紧张 | 中等紧张 | 高紧张 |
|-----|-------|---------|-------|
| 敌人数量 | 1-2个 | 3-5个 | 多个+追踪 |
| 移动速度 | 慢 | 中等 | 快 |
| 空间大小 | 开阔 | 中等 | 狭窄 |
| 音效 | 平静BGM | 紧张BGM | 心跳+警报 |

### 2.3 解谜设计

**谜题类型**：

| 类型 | 描述 | 难度范围 | 参考作品 |
|-----|------|---------|---------|
| 环境解谜 | 利用场景物件解决 | 简单-中等 | 《锈湖》系列 |
| 记忆解谜 | 记忆碎片重组 | 简单-中等 | 《寻找天堂》 |
| 时序解谜 | 特定操作序列 | 中等-困难 | 《锈湖：逃离方块》 |
| 叙事解谜 | 通过对话/收集理解真相 | 简单 | 《空洞骑士》 |
| 观察解谜 | 发现隐藏元素 | 简单 | 《锈湖》系列 |

**解谜与叙事的结合**：

- 谜题答案往往与角色背景故事相关
- 解谜过程揭示游戏世界的隐藏信息
- 完成谜题解锁新的剧情片段
- 部分谜题需要特定情绪状态才能解决

### 2.4 敌人行为模式

**敌人类型（非战斗设计）**：

| 类型 | 行为特点 | 应对方式 | 参考作品 |
|-----|---------|---------|---------|
| 巡逻型 | 固定路线来回移动 | 观察规律、趁机通过 | 《小小梦魇》 |
| 追踪型 | 发现玩家后追击 | 逃跑、躲藏、绕路 | 《小小梦魇》 |
| 潜伏型 | 静止不动，发现后攻击 | 小心观察、避免触碰 | 《空洞骑士》 |
| 感应型 | 对声音/光线敏感 | 保持安静、避免光源 | 《锈湖》系列 |

**敌人与情绪投射**：

- 不同敌人代表不同的负面情绪
- 恐惧情绪 → 追逐型敌人
- 孤独情绪 → 潜伏型敌人
- 焦虑情绪 → 巡逻型敌人
- 面对敌人不是消灭，而是理解与接纳

### 2.5 医学元素设计原型

> 【来自团队讨论】利用医学知识作为技能和怪物设计的原型，可以为游戏带来独特而深刻的体验。陌生的医学名词能够转化为富有想象力的游戏元素。

#### 设计原则

**核心理念**：将医学概念转化为游戏机制时，保留其本质特征，同时赋予奇幻化的表现形式。

**转化方式**：
- **疾病机制** → 异常状态 / 负面效果
- **病原生物** → 敌人设计 / 怪物原型
- **心理知识** → 技能效果 / 角色能力
- **医学术语** → 技能名称 / 道具命名

#### 疾病作为异常状态

| 医学概念 | 游戏表现 | 效果设计 |
|---------|---------|---------|
| 炎症反应 | 「炎症」状态 | 持续受到伤害，移动时伤害增加 |
| 免疫应答 | 「免疫」增益 | 短时间内免疫特定类型伤害 |
| 过敏反应 | 「过敏」异常 | 接触特定元素时触发剧烈反应 |
| 纤维化 | 「硬化」效果 | 防御提升但移动速度下降 |
| 坏死 | 「坏死」诅咒 | 生命值上限逐渐降低 |

#### 病原生物作为怪物原型

**细菌类敌人**：
- **分裂者**：周期性分裂成两个较弱个体
- **耐药体**：受到同类型攻击后产生抗性
- **共生菌**：与其他敌人形成互利关系

**病毒类敌人**：
- **潜伏者**：进入宿主（其他敌人）体内，强化宿主能力
- **变异体**：战斗中不断改变形态和攻击方式
- **传播者**：攻击玩家后，玩家会感染「病毒」状态

**寄生虫类敌人**：
- **附身者**：控制其他敌人或环境物体
- **吸取者**：持续吸取玩家能量恢复自身
- **孵化者**：体内孕育大量小型敌人

#### 心理知识作为技能设计

**防御类技能**：
| 技能名称 | 心理学概念 | 效果 |
|---------|-----------|------|
| 认知重构 | 认知行为疗法 | 将负面状态转化为正面增益 |
| 情绪隔离 | 心理防御机制 | 暂时免疫情绪类伤害 |
| 正念冥想 | 正念疗法 | 持续恢复理智值，降低恐惧 |

**能力类设计**：
| 技能名称 | 心理学概念 | 效果 |
|---------|-----------|------|
| 投射能力 | 心理投射 | 将自身负面状态转移给环境 |
| 情绪爆发 | 情绪宣泄 | 释放累积的情绪改变场景 |
| 认知重构 | 认知失调理论 | 改变敌人的感知范围和行为 |

**辅助类技能**：
| 技能名称 | 心理学概念 | 效果 |
|---------|-----------|------|
| 共情链接 | 共情能力 | 与队友共享生命值恢复 |
| 心流状态 | 心流理论 | 进入专注状态，技能冷却大幅减少 |
| 自我实现 | 马斯洛需求层次 | 短时间内全属性大幅提升 |

#### 医学术语的游戏化命名

**道具命名示例**：
- 抗生素 → 「净梦之露」（清除负面状态）
- 疫苗 → 「预见之种」（提前预知敌人位置）
- 麻醉剂 → 「沉眠之息」（让敌人暂时静止）
- 肾上腺素 → 「觉醒之血」（提升移动速度）

**能力命名示例**：
- 化疗 → 「净化之光」（净化情绪投射的敌人）
- 免疫疗法 → 「自适应护盾」（免疫特定负面状态）
- 基因治疗 → 「本质重塑」（改变自身属性）
- 心理治疗 → 「心灵手术」（揭开敌人的真相）

> 【设计说明】这些对于普通人来说陌生的医学名词，可以带来更棒的体验——既保留了医学概念的深度，又赋予其奇幻色彩，让游戏世界更加独特且富有内涵。
