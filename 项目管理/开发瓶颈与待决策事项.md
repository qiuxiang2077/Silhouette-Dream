## 🎮 瓶颈 #1：核心玩法机制（Critical）

### 问题描述

> **核心矛盾**：我们有了完整的剧情和世界观，但"这个游戏要怎么玩？"

目前的设定已经非常丰富：
- ✅ 影子共生系统（依附/游离/消逝）
- ✅ 情绪投射机制（恐惧→怪物、孤独→迷宫）
- ✅ 睡眠阶段系统（清醒期→浅睡期→深睡期→REM期）
- ✅ 医学元素设计（疾病→异常状态、病原→怪物、心理→技能）

**但核心玩法循环仍未确定**，这直接导致：
- Agent A 无法确定程序需求文档的核心系统规格
- Agent C 无法确定美术资源的动作需求（是否需要战斗动画？）
- 无法确定音效需求（战斗音效？环境音效？）
- 整个第二阶段 Agent 协调工作无法推进

---

### 待决策问题清单

#### 1.1 核心体验类型（必须首先确定）

**问题**：玩家在游戏中主要做什么？

- [ ] **选项A：动作战斗导向**
  - 实时战斗，技能释放，成长build
  - 参考：Hades、以撒的结合、元气骑士
  - 优势：玩法深度高，重复可玩性强
  - 挑战：需要设计影子无实体的战斗方式

- [ ] **选项B：探索解谜导向**
  - 环境解谜，记忆收集，剧情推进
  - 参考：Gris、Journey、Inside
  - 优势：与叙事和氛围契合度高
  - 挑战：需要设计有深度的解谜机制

- [ ] **选项C：叙事体验导向**
  - 步行模拟器，互动叙事，情感体验
  - 参考：What Remains of Edith Finch、Dear Esther
  - 优势：最大化剧情和情感冲击力
  - 挑战：游戏性较弱，可能受众有限

- [ ] **选项D：混合机制**（推荐考虑）
  - 探索解谜为主 + 轻度战斗/潜行
  - 参考：Hollow Knight、Ori
  - 优势：平衡玩法深度和叙事体验
  - 挑战：需要精心设计两种机制的融合

#### 1.2 战斗系统（如果选择包含战斗）

**问题**：如果要战斗，怎么战斗？

**战斗类型**：
- [ ] 实时动作战斗（类Hades）
- [ ] 回合制战斗（类JRPG）
- [ ] 混合机制（实时移动+回合技能）
- [ ] 其他创新机制

**操作方式**：
- [ ] 键盘/手柄操作方案
- [ ] 技能释放方式（按键组合？技能轮盘？）
- [ ] 闪避/防御机制（影子如何闪避？）

**战斗节奏**：
- [ ] 快节奏爽快型（高伤害，快速击杀）
- [ ] 慢节奏策略型（需要观察，寻找破绽）
- [ ] 随睡眠阶段变化（浅睡期慢，REM期快）

#### 1.3 影子战斗方式（世界观一致性）

**问题**：影子没有实体，如何与敌人互动？

- [ ] **方案A：依附物体战斗**
  - 影子依附于梦境中的物体（武器、道具）进行战斗
  - 优势：符合影子特性，有创意
  - 挑战：需要设计物体获取和切换机制

- [ ] **方案B：情绪能量攻击**
  - 影子使用情绪能量进行远程/范围攻击
  - 优势：与情绪投射机制结合
  - 挑战：需要设计能量管理系统

- [ ] **方案C：环境互动**
  - 利用梦境环境（光影、记忆碎片）进行"战斗"
  - 优势：独特，符合梦境主题
  - 挑战：设计难度大，可能不够直观

- [ ] **方案D：概念具象化**
  - 影子在梦境中可以暂时"实体化"
  - 优势：简化设计，传统战斗方式
  - 挑战：与影子无实体的设定有冲突

#### 1.4 睡眠阶段与玩法结合

**问题**：如何将睡眠阶段系统融入核心玩法？

| 睡眠阶段 | 当前设定机制 | 待确定：玩法表现 |
|---------|-------------|-----------------|
| 清醒期 | 路线选择、装备整备 | 是策略规划还是可玩内容？ |
| 浅睡期 | 状态波动、场景重组 | 如何利用不稳定性设计玩法？ |
| 深睡期 | 恐惧累积、资源稀缺 | 是生存挑战还是恐怖体验？ |
| REM期 | 战斗加成、BOSS战 | 如何设计高潮体验？ |

## 🎨 瓶颈 #2：美术风格方向（High Priority）

### 问题描述

> **核心矛盾**：西式奇幻 vs 中国式千禧年梦核风

团队内部对美术风格存在不同偏好，需要在进入正式美术资源制作前确定统一方向。

**当前状态**：
- 一方偏好：西式奇幻（Hollow Knight、Ori风格）
- 另一方偏好：中国式千禧年梦核风（怀旧、本土化）
- 潜在方案：融合风格（梦境=奇幻，现实=梦核）

---

### 待决策问题清单

#### 2.1 整体风格方向

- [ ] **选项A：统一西式奇幻**
  - 梦境和现实都使用西式奇幻风格
  - 参考：Hollow Knight、Ori、Gris
  - 特点：精致的幻想美学、流畅的动画、国际化视觉语言
  - 优势：市场接受度高，技术参考多，团队熟悉

- [ ] **选项B：统一千禧年梦核风**
  - 梦境和现实都使用梦核风格
  - 参考：国产独立游戏的怀旧美学、老式小区、童年记忆
  - 特点：怀旧感、朦胧美、本土化情感共鸣
  - 优势：差异化明显，情感连接强，独特定位

- [ ] **选项C：融合风格**（强烈推荐考虑）
  - **梦境世界**：西式奇幻（超现实、幻想、鲜艳）
  - **现实世界**：千禧年梦核风（怀旧、写实、温暖）
  - 优势：
    - 完美契合双重世界设定
    - 形成强烈视觉对比
    - 兼顾两种偏好
    - 差异化明显
  - 挑战：需要维护两套美术资产

#### 2.2 像素艺术规格

**分辨率标准**：
- [ ] 低像素（320×180，复古感强）
- [ ] 中像素（640×360，平衡选择）
- [ ] 高像素（1920×1080原生，细节丰富）

**色彩方案**：
- [ ] 限制调色板（16-32色，复古统一）
- [ ] 扩展调色板（64-128色，丰富表现）
- [ ] 全色域（现代感，细节丰富）

**调色板方案（如果选融合风格）**：

| 世界 | 主色调 | 强调色 | 氛围 |
|-----|-------|-------|------|
| 梦境 | 紫罗兰、深蓝、金色 | 荧光绿、亮粉色 | 神秘、超现实 |
| 现实 | 米色、棕色、暖灰 | 夕阳橙、暖黄 | 怀旧、温馨 |

#### 2.3 伪3D实现方式

- [ ] **2.5D分层视差**
  - 多层背景+前景，创造深度感
  - 优势：性能友好，效果直观
  - 参考：Ori、Hollow Knight

- [ ] **等距视角**
  - 经典的45度俯视
  - 优势：技术成熟，资源好找
  - 参考：Diablo、Hades

- [ ] **其他创新方案**
  - 根据风格定制
  - 优势：独特，可能形成特色
  - 挑战：技术风险

---

### 决策辅助信息

#### 风格对比分析

| 维度 | 西式奇幻 | 千禧年梦核 | 融合风格 |
|-----|---------|-----------|---------|
| 独特性 | 中（常见） | 高（差异化） | 极高（独特） |
| 情感共鸣（国内） | 中 | 高 | 高 |
| 技术难度 | 低（参考多） | 中 | 高（两套资产） |
| 与设定契合 | 中 | 中（仅现实） | 极高 |
| 市场接受度 | 高 | 中 | 高 |
| 开发成本 | 低 | 中 | 高 |

#### 推荐方案：融合风格

**理由**：
1. **完美契合设定**：梦境=奇幻，现实=梦核，与双重世界设定完全一致
2. **视觉叙事**：风格对比本身就能传达"梦境vs现实"的主题
3. **兼顾双方**：不需要在两种偏好之间妥协
4. **差异化**：市场上几乎没有采用这种对比策略的游戏

**实施建议**：
- 梦境：使用西式奇幻，强调超现实、鲜艳、流动感
- 现实：使用千禧年梦核，强调怀旧、温暖、静止感
- 过渡：设计独特的"入梦"和"醒来"视觉过渡效果
