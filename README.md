# 《影梦》 - 一个伪3D俯视角像素肉鸽冒险游戏

## 🎮 项目概述

**游戏类型**：伪3D俯视角像素风格肉鸽冒险游戏  
**世界观基底**：「梵天一梦」式梦境逻辑，融合现实成长叙事与共生设定  
**参考游戏**：《元气骑士》《饥荒》《喵喵启示录》《星露谷物语》《杀戮尖塔》《土豆兄弟》 
**核心循环**：梦境冒险（局内随机成长） + 现实成长（局外永久解锁）  
**美术风格**：伪3D俯视角像素风，梦境奇幻 vs 现实写实  
**游戏引擎**：Godot 4.x  
**开发状态**：概念设计阶段  
**暂定标题**：《影梦》

### 世界观概述
- **双重世界**：表层是剑与魔法+科幻融合的梦境冒险世界；底层真相是爱幻想小男孩的梦，影子是共生种族
- **现实-梦境联动**：小男孩入睡后影子进入梦境战斗，怪物是现实经历/情绪的投射
- **影子种族设定**：依靠光而存在的古老种族。远古时期太阳永不落下，影子一族繁荣昌盛；时代迁移，太阳衰败导致黑夜诞生，影子一族落入黑夜便会逐渐失去自我、消逝在黑暗中
- **核心主题**：内心成长与自我和解，通过梦境冒险反映现实心理蜕变

### 关键角色
- **影子（主角）**：小男孩的影子，在梦境世界中冒险。白天探索成长，黑夜在无尽挑战中维持自我存在
- **魔女**：幽灵形态的神秘引导者，原型是梦主（小男孩）的妈妈。端庄优雅却傲娇，隐藏着柔软的内心。她编造谎言引导主角，真实动机是寻找生前的家、了解父母对自己的真实想法

### 核心玩法特色
- **昼夜双模式**：
  - **白天（剧情模式）**：探索、对话、推进主线，展现小男孩白天的生活和经历
  - **黑夜（肉鸽模式）**：无尽挑战，面对记忆与情绪化作的假想敌，在黑暗中维持自我存在
  - **切换动画**：点击开始游戏时，画面从白天切换成夜间，象征从现实进入梦境
- **梦境冒险**：伪3D俯视角肉鸽战斗，随机生成梦境关卡，局内拾取武器/道具/技能
- **现实叙事**：小男孩日常互动（校园、家庭、情绪变化），影响梦境内容和难度
- **影子共生**：影子作为主角在梦境中的化身，依靠光而存在，能力随现实成长而进化
- **魔女引导**：每次失败后在魔女处重生，随着冒险逐渐揭开她的谎言与真相
- **情绪投射**：怪物和BOSS是现实负面情绪（如霸凌）的具象化
- **多结局系统**：普通结局 + 魔女真结局（理解并阻止魔女，在故乡阳光下告别）

## 🔧 技术实现要点

### 引擎与工具
- **主引擎**：Godot 4.x（2D引擎对像素游戏支持优秀）
- **美术工具**：Aseprite / Pixelorama
- **版本控制**：Git + GitHub
- **项目管理**：Trello看板 + 飞书文档

### 关键实现设置
1. **像素美术**：纹理导入Filter → Nearest，关闭Mipmaps，启用像素对齐
2. **场景结构**：节点化设计，分离UI层、世界层、实体层
3. **武器系统**：基类继承模式，支持热插拔切换
4. **数据持久化**：Resource格式保存永久进度
